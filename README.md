# Product Management System (Django)

A web-based **Product Management System** developed using **Django** that allows administrators to manage products efficiently.  
The system supports adding, updating, viewing, and deleting products with a user-friendly interface.

This project is suitable for **academic submission**, **Django learning**, and **basic e-commerce / inventory management concepts**.
